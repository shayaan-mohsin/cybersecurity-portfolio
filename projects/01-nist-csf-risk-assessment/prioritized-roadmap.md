# Prioritized Roadmap

## Phase 1: Governance Foundation

- Assign owners for access management, vendor access, incident response, and asset inventory.
- Create a simple security policy register.
- Define how risks are reviewed and accepted.

## Phase 2: Access And Vendor Controls

- Implement quarterly access reviews.
- Require named vendor accounts where possible.
- Remove stale or shared accounts.
- Document least-privilege expectations.

## Phase 3: Incident Response Readiness

- Create a lightweight incident response playbook.
- Define escalation contacts.
- Draft internal and external communication steps.
- Run a tabletop exercise.

## Phase 4: Awareness And Recovery

- Publish security awareness guidance.
- Define phishing reporting steps.
- Document backup expectations.
- Test recovery for one critical workflow.

## Success Measures

- Access reviews completed each quarter
- Asset inventory owner assigned
- Vendor account list reviewed
- Incident response playbook approved
- Tabletop exercise completed
