# Project 01: NIST CSF Risk Assessment

## Objective

Build a fictional GRC and cyber risk assessment using NIST Cybersecurity Framework 2.0 concepts.

## Scenario

Fictional organization: **CareBridge Health Services**

CareBridge is a small healthcare services organization that handles patient scheduling, billing support, employee accounts, and third-party vendor access. The scenario is fictional and designed for portfolio use only.

## Deliverables

- [Scope and assumptions](scope.md)
- [Risk register](risk-register.md)
- [NIST CSF mapping](csf-mapping.md)
- [Prioritized roadmap](prioritized-roadmap.md)
- [Executive brief](executive-brief.md)

## Skills Demonstrated

- GRC analysis
- Risk identification and prioritization
- Framework-based thinking
- Executive communication
- Healthcare security and privacy awareness

## Portfolio Value

This project shows how cybersecurity risk can be translated into business-facing recommendations without overclaiming enterprise ownership or live production access.
