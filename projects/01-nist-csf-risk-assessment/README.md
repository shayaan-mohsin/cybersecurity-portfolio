# Project 01: NIST CSF Risk Assessment

## Objective

Build a recruiter-ready GRC and cyber risk assessment using NIST Cybersecurity Framework (CSF) 2.0 concepts.

## Scenario

Fictional organization: **CareBridge Health Services**

CareBridge is a fictional small healthcare services organization that handles patient scheduling, billing support, employee accounts, cloud storage, and third-party vendor access. The scenario is designed for public portfolio use and does not use real patient data, real employer systems, or copied coursework.

## Assessment Question

How can a small healthcare services organization improve cybersecurity governance, access control, vendor oversight, incident readiness, and resilience using a practical NIST CSF 2.0-aligned roadmap?
